一些哔哩哔哩的 API

2022-04-19 2 min read
  • 登录的 URL: https://passport.bilibili.com/ajax/miniLogin/minilogin
  • 重定向到 https://passport.bilibili.com/ajax/miniLogin/redirect 则为登录成功
  • 取回的 Cookie 中的 bili_jct 即为 csrf
  • 点赞:
  • URL: https://api.bilibili.com/x/web-interface/archive/like
  • 类型: POST
  • 需要 Cookie: 是
  • 返回值类型: JSON
  • 参数: aid (稿件 AV 号)、like (1 为点赞，2 为取消点赞)、csrf (见上)
  • 个人信息获取:
  • URL: https://api.bilibili.com/x/web-interface/nav
  • 类型: GET
  • 需要 Cookie: 是
  • 返回值类型: JSON
  • 参数: 无
  • 搜索:
  • URL: https://api.bilibili.com/x/web-interface/search/type
  • 类型: GET
  • 需要 Cookie: 否
  • 返回值类型: JSON
  • 参数: keyword (关键词)、search_type (video)、page (分页)、order (排序方式)
    Order 参考: default (综合)、pubdate (发布日期倒序)、senddate (修改日期倒序)、id (投稿 ID 倒序)
  • 评论:
  • URL: https://api.bilibili.com/x/v2/reply/add
  • 类型: POST
  • 需要 Cookie: 是
  • 返回值类型: JSON
  • 参数: oid (稿件 AV 号)、message (UTF-8 要发送的文本)、csrf (见上)
