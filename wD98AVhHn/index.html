在 Android 9.0 + 使用 HTTP 请求

2022-03-05 Android ， Java
  • 为了安全，在 Android 9.0 + 之后 Google 禁止了所有非 HTTPS 请求。会报错 not permitted
  • 直接在 AndroidManifest.xml 的 <application> 里插入
    android:usesCleartextTraffic="true"
    就可以正常使用非 HTTPS 请求了
